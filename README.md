`Tablite` is a lightweight database GUI build on tauri.

## Features

- Lightweight and cross platform build
- Natural language querying
- Muli-LLM support
- Modern and easy to use interface
- Privacy with all your data locally

Supported databases:

- SQLite
- MySQL
